# 📝 Python To-Do App

A simple and beginner-friendly **To-Do List Application** built using Python.
This project helps users manage daily tasks efficiently through a clean and interactive command-line interface.

## 🚀 Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Simple and easy-to-use interface
* Lightweight and fast

## 🛠️ Technologies Used

* Python 3

## 📂 Project Structure

```bash
todo-app/
│── todo.py
│── README.md
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/todo-app.git
```

2. Navigate to the project folder:

```bash
cd todo-app
```

3. Run the Python file:

```bash
python todo.py
```
